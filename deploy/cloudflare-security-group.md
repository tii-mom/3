# 腾讯云安全组：仅允许 Cloudflare 回源

源站 `api.3api.shop` 已橙云代理。为防止攻击者绕过 Cloudflare 直连 VPS `80/443`，请在腾讯云安全组收紧入站规则。

官方 IP 列表（请定期核对）：

- IPv4: https://www.cloudflare.com/ips-v4
- IPv6: https://www.cloudflare.com/ips-v6

## 推荐入站规则

| 优先级 | 协议 | 端口 | 来源 | 说明 |
|--------|------|------|------|------|
| 高 | TCP | 22 | 你的办公/家庭公网 IP | SSH（不要 `0.0.0.0/0`） |
| 高 | TCP | 80,443 | 下方 Cloudflare IPv4 段 | HTTP/HTTPS 回源 |
| 可选 | TCP | 80,443 | 下方 Cloudflare IPv6 段 | 若实例开了 IPv6 |
| 默认 | ALL | ALL | 拒绝 | 其它入站拒绝 |

**不要**对公网开放 `5432`、`6379`、`8085`（应用已绑定 `127.0.0.1:8085`）。

## 腾讯云控制台操作步骤

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/) → 云服务器 → 安全组  
2. 找到绑定到 `43.167.220.227` 的安全组 → 入站规则 → 修改  
3. 删除 `0.0.0.0/0` 对 `80/443` 的放行（若存在）  
4. 按上表逐条添加 Cloudflare 网段  
5. 确认 SSH 来源仅为你的 IP 后保存  

## 验证

```bash
# 经 Cloudflare 应成功
curl -sS -o /dev/null -w '%{http_code}\n' https://api.3api.shop/

# 直连源站 IP（收紧后应超时/拒绝；收紧前可能仍通）
curl -sk --resolve api.3api.shop:443:43.167.220.227 https://api.3api.shop/ -o /dev/null -w '%{http_code}\n' --max-time 5
```
